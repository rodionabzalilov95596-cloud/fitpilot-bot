## Адаптер VK (long polling)

Бот работает через **Bots Long Poll API** сообщества ВКонтакте — удобно для тестов с десктопа без публичного webhook.

### Настройка сообщества

1. Создайте сообщество ВК и включите **Сообщения сообщества**.
2. В разделе **Работа с API** → **Ключи доступа** создайте ключ с правами:
   - управление сообществом
   - сообщения сообщества
3. В **Bots Long Poll API** включите long poll и отметьте событие `message_new`.
4. Сохраните токен в `.env` или `.env.txt` (можно просто строку `vk1....`).

### Переменные окружения

```env
VK_BOT_TOKEN=vk1.xxx
VK_GROUP_ID=123456789   # опционально, иначе определяется автоматически
VK_API_VERSION=5.199
```

### Запуск

```bash
npm run dev
```

Если задан `VK_BOT_TOKEN`, при старте поднимается long polling. Локальный эмулятор `POST /simulate` остаётся доступен.

### Формат кнопок

Кнопки сценария (`OutgoingMessage.buttons`) конвертируются в клавиатуру VK. Нажатие возвращает `payload` с `buttonId`, который попадает в `IncomingMessage.buttonId`.
